# Retail Store Analytics Dashboard

## Project Overview

This project is an end-to-end Retail Store Analytics Dashboard developed using Power BI. It provides interactive reports to analyze sales performance, customer behavior, product performance, and business profitability. The dashboard is designed to help stakeholders monitor key business metrics and make informed decisions through data visualization.

## Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- SQL (MySQL)
- Microsoft Excel / CSV

## Dashboard Pages

### Executive Summary
- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Sales Trend
- Sales by Category
- Sales by Region
- Top 10 Products by Sales & Profit

### Customer Analysis Dashboard
- Total Customers
- Average Order Value
- Profit per Customer
- Sales by Customer Segment
- Profit by Customer Segment
- Customer Profitability
- Regional Customer Analysis

### Product & Category Analysis
- Total Products
- Category-wise Profit
- Sales by Category
- Sales by Sub-Category
- Top 10 Loss-Making Products
- Product Performance Analysis

### Profitability & Operations Dashboard
- Total Profit
- Profit Margin
- Average Discount
- Total Shipping Cost
- Profit by Region
- Profit by Market
- Discount vs Profit Analysis
- Profit by Order Priority
